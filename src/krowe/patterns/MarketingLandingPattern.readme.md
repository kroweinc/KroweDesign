# MarketingLandingPattern

Source: `src/krowe/patterns/MarketingLandingPattern.tsx`.

## Purpose

Full-page **marketing landing** composition: hero with motion, stats strip, footer. Uses `BrandMark`, `useInView`, `useCountUp`.

## Usage

```tsx
import { MarketingLandingPattern } from '@/krowe/patterns/MarketingLandingPattern';

export function LandingPage() {
  return <MarketingLandingPattern />;
}
```

No props — **demo/reference layout**. Copy patterns into a routed page and replace copy and links for production.
