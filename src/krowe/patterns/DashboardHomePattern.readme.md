# DashboardHomePattern

Source: `src/krowe/patterns/DashboardHomePattern.tsx`.

## Purpose

**Dashboard home** with `Sidebar`, featured idea card, grid of ideas, suggested next steps. Uses `useMouseTilt` on a highlight card.

## Usage

```tsx
import { DashboardHomePattern } from '@/krowe/patterns/DashboardHomePattern';

<DashboardHomePattern />
```

Adjust `marginLeft: '260px'` if your sidebar width changes.
