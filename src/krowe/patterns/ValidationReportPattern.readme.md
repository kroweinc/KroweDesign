# ValidationReportPattern

Source: `src/krowe/patterns/ValidationReportPattern.tsx`.

## Purpose

App-shell **validation report**: `Sidebar`, `ContentHeader`, `VerdictCard`, collapsible sections with `MarketSizeChart`, `CompetitorScatter`, `MVPCostBar`, motion scroll effects.

## Usage

```tsx
import { ValidationReportPattern } from '@/krowe/patterns/ValidationReportPattern';

<ValidationReportPattern />
```

Replace `REPORT_SECTIONS` data with API-driven content; keep layout primitives.
